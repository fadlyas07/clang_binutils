# Backed Prebuilt Clang with Binutils

## Repository Objective

This repository provides prebuilt Clang toolchains integrated with Binutils. Sources may come from [AOSP Clang](https://android.googlesource.com/platform/prebuilts/clang/host/linux-x86) or official [LLVM releases](https://github.com/llvm/llvm-project/releases).
